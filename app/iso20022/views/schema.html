ISO 20022 Schema

Below tools will assist you to understand this complex standard in your financial messaging systems.

Tools You'll Need

  1. ISO 20022 XSD (XML Schema Definition)

    The ISO 20022 XSD defines the structure of ISO 20022 messages. You can download the necessary XSD files from the official ISO 20022 website or other trusted sources.

  2. xsd-json-converter

    Use the 'xsd-json-converter' to convert the XSD files to JSON Schema. This simplifies the process of working with the data programmatically and with other applications.

  3. XSDService

    The powerful dotnet library which transform complex XSD to minimal JSON structure. This library will help user to deconstruct complex XSD to JSON which further help to understand the structure and able to construct more important things around it.
